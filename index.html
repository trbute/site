Life - A JavaScript implementation of Conway's Game of Life.

A grid-based cellular automaton where each cell exists in one of two states - alive or dead.

For each generation, the state of each cell is determined by the following rules:
  • Each living cell with one or no neighbors dies, as if by solitude.
  • Each living cell with four or more neighbors dies, as if by overpopulation.
  • Each living cell with two or three neighbors survives.
  • Each dead cell with three neighbors becomes populated.
The can be boiled down to two rules that define the criteria for each state to change:
  • Living cells die with less than 2 or more than 3 living neighbors.
  • Dead cells become alive with exactly 3 living neighbors.
This implementation uses HTML 5 Canvas to draw the grid and game state.
Cells can be drawn or removed by left or right-clicking on the canvas.
Middle mouse click and scroll wheel allow for pan and zoom.
Various on-screen controls allow for navigating through the game state.
